Nice, the whole RAG flow works.

Now that we can search a workspace, we're going to train our own adapter.

Fine-tuning takes a base model and produces a small LoRA (low-rank adaptation) file alongside it. The adapter is a small file with the trained weight deltas; the base model stays untouched. Loading the base model with `modelConfig.lora` pointing at the adapter file routes completions through the trained behavior.
