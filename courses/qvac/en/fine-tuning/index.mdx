Nice, the whole RAG flow works.

Now that we can search a workspace, we're going to train our own adapter.

Fine-tuning takes a base model and produces a small LoRA (low-rank adaptation) file alongside it. The adapter is a small file with the trained weight deltas; the base model stays untouched. At inference time we hand the adapter back to `loadModel()` and the resulting completions reflect the trained behavior.
