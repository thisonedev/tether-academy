Now that we can make images, we're going to make them move.

Video diffusion builds on the same engine as image diffusion, with a few extra files (the T5-XXL text encoder and a VAE tuned for frames) and a `mode: "video"` flag. The result is an AVI file of frames at the rate we asked for.
