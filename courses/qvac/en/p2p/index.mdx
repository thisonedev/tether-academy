Now that we've covered all the model surfaces, we're going to look at how the models actually get to your machine.

The peer-to-peer (or P2P) surface is about model distribution. QVAC pulls model files from a peer-to-peer Hyperdrive instead of a single CDN, which means downloads can come from whoever has the file already and can serve as a seeder. This chapter covers the two patterns you'll use most often: pre-downloading a model and configuring blind relays for NAT traversal.
