BCI models are a much more unusual capability, the brain-computer interface.

BCI models take a neural signal buffer and produce a text transcription. The output plugs into the same completion / event APIs the rest of the SDK uses. The shape is the same: load a model, hand it input, read back the structured result. The inputs and outputs are unusual, but the workflow isn't.
